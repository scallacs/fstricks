Frequently Ask Question

Adding new tricks

You can add new tricks from any supported video provider in two steps.
First you must add the video here.

Then you can add one or more trick to this video. You must precise the start and end time and the name of the trick.
Be as precise as possible. The better your trick is the better it's going to be vote up by others.

What if there is an invalid trick

You can use the report error form to do so. You can add a message to explain why it's not a good trick. It can be for several reasons, among them the name of the trick is not good, the begin or the end time or also a too poor video quality.
