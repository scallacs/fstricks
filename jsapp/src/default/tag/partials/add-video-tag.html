How it works

If you have seen a trick that you like in a video, you can register it here. You just have to pick the time range for the trick (see rules below). Then you fill in the trick and rider name and click on the "Add" button. That's it! It will now be validated by other members before being visible to everybody.

In the meantime, you still can edit or remove the trick if you like.

Rules

One trick at a time

Each time range should contain only one trick at a time.

Time range from the beginning to the end of the trick

It's important to be accurate on the time range. You must set the start time a few seconds before the beginning of the trick. It's better if it's in the same video shot.
The end time must be a few seconds after the landing. If there are multiple shots or replay in raw, include them all.

Be accurate

When you fill the rider name and trick name be as accurate as possible. Use existing riders and tricks in priority. If you need to create a new one, do not use abbreviations, use the full name.
